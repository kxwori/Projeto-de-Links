# Projeto Links

Este é um mini projeto pedagógico desenvolvido em HTML e CSS, estruturado com o objetivo prático de aplicar conceitos essenciais de controle de versão utilizando Git e GitHub.

## Objetivo da Atividade
O projeto simula um fluxo real de desenvolvimento de software, cobrindo os seguintes tópicos solicitados na atividade do Professor Marcão:
- Criação e versionamento de um código base (HTML/CSS).
- Gerenciamento de tarefas utilizando **GitHub Issues**.
- Trabalho paralelo com criação de **Branches** distintas (`feat-link-git` e `style-dark-mode`).
- Integração de código através de **Pull Requests (PRs)**.

## Tecnologias Utilizadas
- HTML5
- CSS3
- Git & GitHub

## Funcionalidades e Fluxos Implementados
1. **Estrutura de Favoritos:** Uma página simples centralizando links úteis de documentação.
2. **Resolução de Issues:** Adicionado dinamicamente o link da documentação do Git referenciando e fechando a Issue #1.
3. **Tema Escuro:** Implementação de uma folha de estilo voltada para o modo escuro a fim de praticar ramificações de código paralelas.
